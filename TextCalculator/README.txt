* This is an application which is developed using ASP.NET Core and Vanila JS. Simple text calculator which is recieving some input of numbers in string and is returning sum of the respective numbers or returning an error.
*Every scenario of the appliacation is tested using XUnit framework.

*Steps on how to open and test the application:
1. Open the file named 'Task2.TextCalculator.sln' in Visual studio from the folder named 'Task2'.
2. Run the WebApi, and leave it open in background.
3. Now from the folder named 'TextCalculatorFrontEnd' open the file index.html.
4. Use the application.
